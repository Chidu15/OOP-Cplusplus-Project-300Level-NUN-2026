Project Name: Personal Diary App
Authors: Ihenacho Chidubem 20231660, Ehiagwina Joachim 20231811, Enechi Sonia 20231027 
Date: 2026-01-21

Description:
A secure, daily diary application built with C++ and Qt 6. It features password protection and a calendar interface for managing daily entries.

How to Compile:
1. Ensure CMake and MinGW (Qt 6.10.1) are installed.
2. Open the project folder in VS Code.
3. Configure the project using the CMake Tools extension.
4. Build and Run.

How to Run:
- The executable is located in the 'build' directory.
- On first launch, you will be prompted to create a password.
- Subsequent launches require this password to access your diary.

Features:
- Encrypted Password Storage
- Date-based file storage (one .txt file per day)
- GUI Navigation

Dependencies:
- Qt 6 Widgets Module
- CMake 3.16+
